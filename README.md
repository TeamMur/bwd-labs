# Лабораторные по Основам веб-разработки
Юганов Тимур ПрИ-22

Указания:
Шаблон названий коммитов:
  {Тип коммита}: {Номер задачи}
  <Описание с ответом на вопрос “Что сделано?”>
  Пример: feat: LAB4-1 Добавлено Readme с описанием шагов для запуска приложения
  Важно! Один коммит включает в себя максимум одну задачу
  
Также, каждая задача должна быть реализована в отдельной ветке, поэтому для веток в дальнейшем примем следующий шаблон названий:
  {Тип ветки}/{Номер задачи}
  Пример: feature/LAB4-1
  Важно! Одна ветка включает в себя максимум одну задачу и должна ветвится от основной ветки (main или master)
